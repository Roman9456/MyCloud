# Frontend README 

## Frontend - Registration and Authentication

This repository contains the frontend for the Registration and Authentication system. It allows users to sign up, log in, and interact with the backend for authentication purposes.

## Technologies Used
- React
- Axios (for HTTP requests)
- React Router (for navigation)
- CSS for styling
